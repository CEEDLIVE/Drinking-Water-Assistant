## 기능 
1. 잠금화면
2. FCM
3. 그동안 마신 물의 History
4. 하루동안 마신 물의 History
5. 물 마시는 단위 변경 가능
6. 24시마다 Today Water 변경

## 1.1.0 update 예정 기능
1. Alarm 기능
2. 물 마시는 단위 Custom
3. FCM 각 화면별 Push가능하도록 및 Push 설정 스위치
